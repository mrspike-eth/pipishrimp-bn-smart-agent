## 🚀 快速启动

```bash
# 克隆仓库
git clone https://github.com/mrspike-eth/Wali-Smart-Agent-based-on-Binance.git

# 进入目录
cd Wali-Smart-Agent-based-on-Binance

# 安装依赖
pip3 install -r requirements.txt

# 配置环境
source config/env.sh

# 启动 Wali
./start.sh
```
